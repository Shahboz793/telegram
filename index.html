Platforma haqida

Bu Ta'lim Edu uchun HTML + CSS + JS + Firestore asosida yaratilgan kurs tizimi.

• Admin — kurs yaratadi, darslar qo‘shadi, foydalanuvchilarni ko‘radi.
• Foydalanuvchi — kiradi, kurslarni ko‘radi, darslarni tomosha qiladi, kursga izoh va baho qoldiradi.

Foydalanuvchi darslarni ko‘rib borgan sari kurs % bo‘yicha tugallanishi hisoblanadi. 100% bo‘lsa maxsus tabrik chiqadi.
